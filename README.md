# Clinic-Medical-Reservation-System
Clinic Medical Reservation System is a production-style backend application designed to manage clinic operations, including appointment booking, doctor schedules, patient management, branch management, and specializations.

The system follows Clean Architecture principles and implements modern backend development practices such as CQRS, MediatR, JWT Authentication, FluentValidation, Caching, Background Services, and Docker containerization.

Patients can browse doctors, view available schedules, and reserve appointments. Doctors can manage their schedules and appointments through a request-based workflow, while administrators have full control over branches, doctors, schedules, appointments, and specializations.

The project was built as a portfolio project to demonstrate real-world .NET backend development skills and production-ready architecture patterns.
